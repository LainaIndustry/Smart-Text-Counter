Smart Text Counter
A responsive web tool for real-time text analysis with word, character, sentence, and paragraph counting.

Features
Real-time text statistics

Reading level analysis

Time estimations (reading & speaking)

Text editing tools (Clear, Undo, Redo)

Fully responsive design

Multiple pages: Home, About, Terms, Privacy, Contact

Tech Stack
HTML5, CSS3, JavaScript

Font Awesome icons

Flexbox/Grid layouts

Usage
Open index.html in a browser

Type or paste text into the editor

View real-time statistics

Files
index.html - Main tool interface

about.html - About information

terms.html - Terms of Service

privacy.html - Privacy Policy

contact.html - Contact form

style.css - All styling

script.js - Functionality

Browser Support
Works on all modern browsers (Chrome, Firefox, Safari, Edge)

No installation required - just open and use!
